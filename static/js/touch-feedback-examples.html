Sistema de Touch Feedback - Exemplos

Demonstração das funcionalidades do sistema de feedback touch para dispositivos móveis.

1. Ripple Effect em Botões

Clique ou toque nos botões abaixo para ver o efeito ripple. O efeito é aplicado automaticamente a todos os botões.

2. Prevenção de Duplo Clique

Botões de ação são automaticamente protegidos contra duplo clique. Tente clicar rapidamente múltiplas vezes - apenas o primeiro clique será processado.

Os botões ficam desabilitados por 300ms após o clique e mostram um spinner.

5. Integração com Formulários

Botões de submit em formulários são automaticamente protegidos.

6. API JavaScript

Você pode controlar o sistema programaticamente. 

// Refresh o sistema após adicionar conteúdo dinâmico
window.touchFeedback.refresh();

// Resetar um botão específico
window.touchFeedback.resetButton(buttonElement);

// Resetar todos os botões
window.touchFeedback.resetAllButtons();

7. Configuração Personalizada

Você pode personalizar o comportamento do sistema. 

// Inicializar com configurações personalizadas
const customTouchFeedback = new TouchFeedback({
    rippleDuration: 800,        // Duração do ripple (ms)
    debounceTime: 500,          // Tempo de debounce (ms)
    rippleColor: 'rgba(0, 123, 255, 0.5)'  // Cor do ripple
});

// Ou reinicializar o sistema global
window.initTouchFeedback({
    rippleDuration: 800,
    debounceTime: 500
});
